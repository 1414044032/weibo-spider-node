node微博爬虫
===========================

## 环境依赖

node v12.13.0

windows 10

## 部署步骤

1. npm install  //安装依赖包

2. config.js  //配置微博账号，密码，浏览器模式

3. node searh_keyword_spider.js //微博（综合）关键词爬取

   node search_user_spider.js //微博（用户）关键词爬取
   
   node search_article_spider.js //微博（用户）关键词爬取
